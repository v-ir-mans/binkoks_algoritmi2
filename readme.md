# Binārais meklēšanas koks
---
## Darba nodošana

Izveidot BMK, realizējot darbības:

- **Koka saknes veidošana un izvadīšana**  
- **Elementa pievienošana**  
- **Elementa meklēšana**  
- **Elementa dzēšana**  
- **Koka izdrukāšana**  
- **Koka apstaigāšana** (preorder, inorder, postorder)  
- **Elementu skaita noteikšana**  
- **Koka pagriešana**
